# 7. 불 자료형

## 7.1 불 자료형이란?
- bool 자료형이란 참과 거짓을 나타내는 자료형이다.
- True와 False는 파이썬의 예약어로 대문자로 작성해야 한다.

## 7.2 불 자료형 선언
```python
a = True
b = False
```

## 7.3 불 자료형 활용
```python
1 == 1
True

2 > 1
True

2 < 1
False
```

## 7.4 자료형의 참과 거짓
- "python" : 참
- \[1, 2, 3\]: 참
- (1, 2, 3): 참
- {'a': 1}: 참
- "": 거짓
- []: 거짓
- (): 거짓
- {}: 거짓
- 1: 참
- 0: 거짓
- None: 거짓
```python
a = [1, 2, 3, 4]
while a:
    print(a.pop())

4
3
2
1
```
