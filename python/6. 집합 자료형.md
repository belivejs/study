# 6. 집합

## 6.1 집합이란
- Set은 집합에 관련된 것을 쉽게 처리하기 위해 만든 자료형이다.
- 중복을 허용하지 않는다.
- 순서가 없다. 순서가 없기 때문에 인덱싱이나 슬라이싱을 할 수 없다.

## 6.2 집합 자료형 선언
```
s1 = set([1, 2, 3])
s1
{1, 2, 3}
```
```
s2 = set("Hello")
s2
{'e', 'H', 'l', 'o'}
```
```
s = set()
```

## 6.3 교집합, 합집합, 차집합
```
s1 = set([1, 2, 3, 4, 5, 6])
s2 = set([4, 5, 6, 7, 8, 9])

# 교집합
s1 & s2
{4, 5, 6}

s1.intersection(s2)
{4, 5, 6}

# 합집합
s1 | s2
{1, 2, 3, 4, 5, 6, 7, 8, 9}

s1.union(s2)
{1, 2, 3, 4, 5, 6, 7, 8, 9}

# 차집합
s1 - s2
{1, 2, 3}

s1.difference(s2)
{1, 2, 3}
```

## 6.4 집합 자료형 관련 함수
- 값 1개 추가하기
    ```
    s1 = set([1, 2, 3])
    s1.add(4)

    s1
    {1, 2, 3, 4}
    ```
- 값 여러 개 추가하기
    ```
    s1 = set([1, 2, 3])
    s1.update([4, 5, 6])

    s1
    {1, 2, 3, 4, 5, 6}
    ```
- 특정 값 제거하기
    ```
    s1 = set([1, 2, 3])
    s1.remove(2)
    
    s1
    {1, 3}
    ```
