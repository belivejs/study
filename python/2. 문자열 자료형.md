
# 2. 문자열
- 문자열을 만드는 방법은 총 4가지이다.
    ```
    "Hello world"
    'Hello world'
    """Hello world"""
    '''Hello world'''
    ```
- 문자열 안에 따옴표를 사용할 경우
    ```
    "Python's favorite food is perl"
    '"Python is very easy." he says.'

    # 이스케이프 문자 사용
    'Python\'s favorite food is perl'
    ```
- 여러 줄인 문자열을 변수에 대입할 경우
    ```
    multiline='''
    Life is too short
    You need python
    '''

    multiline="""
    Life is too short
    You need python
    """
    ```

## 2.1 이스케이프 코드
- \n: 줄 바꿈
- \t: 탭 간격
- \\\\: \을 그대로 표현할 때 사용
- \\': 작은 따옴표를 그대로 표현할 때 사용
- \\": 큰 따옴표를 그대로 표현할 때 사용
- \\r: 캐리지 리턴(줄 바꿈 및 커서를 현재 줄의 가장 앞으로 이동)
- \\f: 폼 피드(줄 바꿈 및 커서를 현재 줄의 다음 줄로 이동)
- \\a: 벨 소리 출력
- \\b: 백 스페이스
- \\000: 널 문자

## 2.2 문자열 연산
- 문자열 더하기
    ```
    head = "Python"
    tail = " is fun!"

    head + tail
    'Python is fun!'
    ```
- 문자열 곱하기
    ```
    a = "python"

    a * 2
    'pythonpython'
    ```

## 2.3 문자열 길이 구하기
```
a = "Life is too short"

len(a)
17
```

## 2.4 문자열 인덱싱
- 파이썬은 0부터 숫자를 센다.
- '-'기호를 붙이면 뒤에서부터 센다.
```
a = "Life is too short, You need Python"

a[3]
'e'
```

## 2.5 문자열 슬라이싱
- 끝 번호에 해당하는 문자는 포함하지 않는다.
```
a = "Life is too short, You need Python"

a[0:4]
'Life'
```

## 2.6 문자열 포매팅
- 문자열 안의 특정한 값을 바꿔야 할 경우가 있을 때 이것을 가능하게 해 주는 것이 문자열 포매팅이다.
- 문자열 포맷 코드
    - %s: 문자열
    - %c: 문자 1개
    - %d: 정수
    - %f: 부동소수
    - %o: 8진수
    - %x: 16진수
    - %%: % 문자 자체
- %s를 사용하면 뒤에 실수가 오든 정수가 오든 모두 문자열로 취급하여 넣는다.
    ```
    "I eat %d apples." % 3
    'I eat 3 apples.'

    "I eat %s apples." % "five"
    'I eat five apples.'

    number = 3
    "I eat %d apples." % number
    'I eat 3 apples.'

    number = 10
    day = "three"
    "I ate %d apples. so I was sick for %s days." % (number, day)
    'I ate 10 apples. so I was sick for three days.'
    ```
- 포팻 코드를 숫자와 함께 사용하면 더 유용하다.
    ```
    # 전체 길이가 10인 문자열 공간, 오른쪽 정렬, 나머지 공백
    "%10s" % "hi"
    '        hi'

    # 왼쪽 정렬은 -를 붙인다.
    "%-10sjane." % 'hi'
    'hi        jane.'

    # 소수점 표현
    "%0.4f" % 3.42134234
    '3.4213'
    ```
- format 함수를 사용한 포매팅 방법
    ```
    "I eat {0} apples".format(3)
    'I eat 3 apples'

    "I eat {0} apples".format("five")
    'I eat five apples'

    number = 3
    "I eat {0} apples".format(number)
    'I eat 3 apples'

    number = 10
    day = "three"
    "I ate {0} apples. so I was sick for {1} days.".format(number, day)
    'I ate 10 apples. so I was sick for three days.'

    "I ate {number} apples. so I was sick for {day} days.".format(number=10, day=3)
    'I ate 10 apples. so I was sick for 3 days.'

    "I ate {0} apples. so I was sick for {day} days.".format(10, day=3)
    'I ate 10 apples. so I was sick for 3 days.'

    # 정렬
    "{0:<10}".format("hi")
    'hi        '

    "{0:>10}".format("hi")
    '        hi'

    "{0:^10}".format("hi")
    '    hi    '

    # 공백 채우기
    "{0:=^10}".format("hi")
    '====hi===='

    "{0:!<10}".format("hi")
    'hi!!!!!!!!'
    ```
- f 문자열 포매팅
    ```
    name = '홍길동'
    age = 30

    f'나의 이름은 {name}입니다. 나이는 {age}입니다.'
    '나의 이름은 홍길동입니다. 나이는 30입니다.'

    age = 30
    f'나는 내년이면 {age + 1}살이 된다.'
    '나는 내년이면 31살이 된다.'

    d = {'name':'홍길동', 'age':30}
    f'나의 이름은 {d["name"]}입니다. 나이는 {d["age"]}입니다.'
    '나의 이름은 홍길동입니다. 나이는 30입니다.'
    ```
