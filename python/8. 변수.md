# 8. 변수

## 8.1 변수란?
- 객체를 가리키는 것
- [1, 2, 3]이라는 값을 가지는 리스트 객체는 메모리에 생성되고, 변수 a는 리스트가 저장된 메모리의 주소를 가리키게 된다.
    ```python
    a = [1, 2, 3]
    
    id(a)
    4303029896
    ```

## 8.2 변수 선언
```python
a = 1
b = "python"
c = [1, 2, 3]
```

## 8.3 리스트 복사하는 경우
- 변수 a는 단지 리스트 객체의 메모리 주소를 가지고 있으므로, 변수 b에는 메모리 주소가 할당된다. 그러므로 b 또한 동일한 리스트 객체를 의미하게 된다.
    ```python
    a = [1, 2, 3]
    b = a

    id(a)
    4303029896

    id(b)
    4303029896
    ```
- 대신 슬라이싱을 사용하여 복사할 수 있다. 슬라이싱을 사용하면 원본 객체와 동일한 타입의 새로운 객체를 반환한다.
    ```python
    a = [1, 2, 3]
    b = a[:]
    a[1] = 4

    a
    [1, 4, 3]
    b
    [1, 2, 3]
    ```
- copy 모듈을 사용해도 된다.
    ```python
    from copy import copy
    a = [1, 2, 3]
    b = copy(a)

    b is a
    False
    ```

## 8.4 변수를 만드는 방법
- 튜플로 할당
    ```python
    a, b = ('python', 'life')
    (a, b) = 'python', 'life'
    ```
- 리스트로 할당
    ```python
    [a, b] = ['python', 'life']
    ```
- 여러개의 변수에 같은 값을 할당
    ```python
    a = b = 'python'
    ```

## 8.5 두 변수의 값을 바꾸기
```python
a = 3
b = 5
a, b = b, a

a
5

b
3
```
