# 17. 모듈

## 17.1 모듈이란?
- 모듈이란 함수나 변수 또는 클래스를 모아 놓은 파이썬 파일이다.
- 파이썬 파일은 모두 모듈이다.

## 17.2 모듈 불러오기
- import는 현재 디렉토리에 있는 파일이나 파이썬 라이브러리가 저장된 디렉토리에 있는 모듈만 불러올 수 있다.
- 확장자는 포함하지 않는다.
- 함수를 불러오면, 모듈 이름 없이 함수 이름만으로 사용할 수 있다.
```python
import 모듈_이름
```
```python
from 모듈_이름 import 모듈_함수

from mod1 import add, sub
from mod1 import *
```
