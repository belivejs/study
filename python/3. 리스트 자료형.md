# 3. 리스트

## 3.1 리스트 선언
```
odd = [1, 3, 5, 7, 9]
```

## 3.2 리스트 인덱싱
```
a = [1, 2, 3]

a[0]
1
```

## 3.3 리스트 슬라이싱
```
a = [1, 2, 3, 4, 5]

a[0:2]
[1, 2]
```

## 3.4 리스트 연산
- 리스트 사이에서 +는 리스트를 합치는 기능을 한다.
    ```
    a = [1, 2, 3]
    b = [4, 5, 6]

    a + b
    [1, 2, 3, 4, 5, 6]
    ```
- 리스트 반복하기
    ```
    a = [1, 2, 3]
    
    a * 3
    [1, 2, 3, 1, 2, 3, 1, 2, 3]
    ```

## 3.5 리스트 값 수정
```
a = [1, 2, 3]
a[2] = 4

a
[1, 2, 4]
```
```
a = [1, 2, 3]
del a[1]

a
[1, 3]
```

## 3.6 리스트 관련 함수
- 리스트에 요소 추가
    ```
    a = [1, 2, 3]
    a.append(4)
    
    a
    [1, 2, 3, 4]
    ```
- 리스트 정렬
    ```
    a = [1, 4, 3, 2]
    a.sort()
    
    a
    [1, 2, 3, 4]
    ```
    ```
    a = ['a', 'c', 'b']
    a.sort()
    
    a
    ['a', 'b', 'c']
    ```
- 리스트 뒤집기
    ```
    a = ['a', 'c', 'b']
    a.reverse()

    a
    ['b', 'c', 'a']
    ```
- 인덱스 반환
    ```
    a = [1, 2, 3]

    a.index(3)
    2

    a.index(1)
    0
    ```
- 요소 삽입
    ```
    a = [1, 2, 3]
    a.insert(0, 4)
    
    a
    [4, 1, 2, 3]
    ```
- 요소 제거
    ```
    a = [1, 2, 3, 1, 2, 3]
    a.remove(3)
    
    a
    [1, 2, 1, 2, 3]
    ```
- 요소 끄집어 내기
    - 리스트의 맨 마지막 요소를 리턴하고 그 요소를 삭제한다.
    ```
    a = [1, 2, 3]
    a.pop()
    3
    
    a
    [1, 2]
    ```
    ```
    a = [1, 2, 3]
    a.pop(1)
    2
    
    a
    [1, 3]
    ```
- 리스트에 포함된 요소 x의 개수 세기
    ```
    a = [1, 2, 3, 1]
    
    a.count(1)
    2
    ```
- 리스트 확장
    ```
    a = [1, 2, 3]
    a.extend([4, 5])
    
    a
    [1, 2, 3, 4, 5]
    ```
