# 4. 튜플

## 4.1 튜플 선언
- 튜플과 리스트의 가장 큰 차이는 요솟값을 변화시킬 수 있는지의 여부이다.
- 요솟값이 변하길 원하지 않는다면 튜플을 사용한다.
    ```python
    t1 = ()
    t2 = (1,)
    t3 = (1, 2, 3)
    t4 = 1, 2, 3
    t5 = ('a', 'b', ('ab', 'cd'))
    ```

## 4.2 튜플 인덱싱
```python
t1 = (1, 2, 'a', 'b')

t1[0]
1

t1[3]
'b'
```

## 4.3 튜플 슬라이싱
```python
t1 = (1, 2, 'a', 'b')

t1[1:]
(2, 'a', 'b')
```

## 4.4 튜플 더하기
```python
t1 = (1, 2, 'a', 'b')
t2 = (3, 4)
t3 = t1 + t2

t3
(1, 2, 'a', 'b', 3, 4)
```

## 4.5 튜플 곱하기
```python
t2 = (3, 4)
t3 = t2 * 3

t3
(3, 4, 3, 4, 3, 4)
```

## 4.6 튜플 길이 구하기
```python
t1 = (1, 2, 'a', 'b')

len(t1)
4
```
